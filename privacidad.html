Politica de privacidad

Esta politica explica como IMPORTADOS RKT utiliza la informacion necesaria para responder consultas y coordinar compras.

Datos que podemos solicitar

Nombre, zona de entrega, datos de contacto, productos consultados, talles y preferencias necesarias para coordinar la compra.

Uso de la informacion

Usamos estos datos solamente para confirmar disponibilidad, coordinar envios o retiros, responder consultas y gestionar cambios.

WhatsApp y plataformas externas

Al finalizar una compra se abre WhatsApp con el resumen del pedido. El uso de WhatsApp, Instagram, TikTok, Mercado Pago o Cuenta DNI depende de sus propias politicas.

Conservacion y seguridad

No vendemos datos personales. Conservamos la informacion solo durante el tiempo necesario para atencion, seguimiento de compra o respaldo administrativo.

Contacto

Para consultar o pedir correccion de datos, escribinos por WhatsApp desde la seccion de contacto.
